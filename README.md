# 🎰 Roulette Analytics PRO

Профессиональный инструмент для анализа статистических аномалий и вероятностей в европейской рулетке. Система отслеживает текущие серии выпадений и рассчитывает риск прерывания аномальных цепочек в реальном времени.

## 🚀 Основные возможности

* **Мониторинг серий (Streaks):** Автоматический подсчет текущей длины серии для цветов, четности, половин и дюжин.
* **Расчет Риска:** Математическая оценка вероятности того, что текущая серия прервется следующим броском (базируется на формуле кумулятивной вероятности).
* **Цветовая индикация аномалий:**
    * 🟢 **< 70%**: Нормальное состояние.
    * 🟡 **70% - 90%**: Внимание, назревает аномалия.
    * 🔴 **> 90%**: Критическое отклонение (высокое ожидание возврата к среднему).
* **Глубокая статистика чисел:**
    * **$\Sigma$ (Sigma):** Общее количество выпадений каждого числа.
    * **$\Delta$ (Delta):** Интервал (количество бросков) с момента последнего появления числа.
* **Удобный интерфейс:** Фиксированная панель ввода, бесконечная история бросков и сортируемая таблица результатов.

## 🛠 Технологический стек

* **Backend:** PHP 8.x
* **Frontend:** HTML5, CSS3 (с поддержкой анимаций и адаптивной сетки Grid)
* **Storage:** PHP Sessions (хранение истории бросков в рамках сессии)

## 📊 Математическая логика

Система использует классическую формулу вероятности для независимых событий. Например, для серии из $n$ событий одного типа с вероятностью $p$, вероятность продолжения серии рассчитывается как $p^{n+1}$. 
**Риск прерывания** вычисляется как $1 - p^{n+1}$, что позволяет наглядно увидеть "натяжение струны" при длительных повторениях.

## 📦 Установка

1. Склонируйте репозиторий или скачайте файлы `index.php`, `functions.php` и `style.css`.
2. Разместите их на локальном сервере (Apache/Nginx) с поддержкой PHP.
3. Откройте `index.php` в браузере.
